## 🛠️ Key Features
- **Role-Based Access Control:** Distinct Admin and User views with restricted capabilities.
- **Product Module:** Advanced data grids with search, multi-category filtering, sorting, and pagination.
- **Analytics Center:** Dynamic visual metrics (Total Inventory Value, Category Distribution) using data processing.
- **URL Synchronization:** Full synchronization of filter states within URL query parameters.
- **Performance Optimized:** Built with debounced search inputs, lazy loading, and memoized components.

## 📦 Tech Stack
- React / Next.js
- Tailwind CSS
- TypeScript
- Recharts (for analytics visualization)
